## Overview

StoreSync Web is a role-based retail analytics dashboard built for store owners to monitor branch performance, track daily financial records, manage managers, and analyze business profitability.

The system provides financial insights including sales, expenses, purchases, branch comparison, and profit analytics.

It demonstrates secure API integration, JWT authentication, role-based access control, and data visualization.

## Authentication Strategy

StoreSync uses:

- JWT tokens generated by backend
- Stored in httpOnly cookies
- Cookies sent automatically via `withCredentials: true`

For testing purposes, the system also supports:
- Authorization header tokens (Bearer)

This ensures both security and testing flexibility.

---

## Dashboard Analytics Logic

The dashboard aggregates:

- Total Sales
- Total Expenses
- Total Purchases
- Net Profit = Sales - (Expenses + Purchases)
- Average Daily Sales
- Branch Comparison (profit ranking)
- Daily Sales Trend

All calculations are performed on backend and visualized on frontend.

## System Flow

1. Owner logs in
2. JWT cookie is stored
3. Dashboard fetches `/api/dashboard`
4. Owner can:
   - Filter by branch
   - Filter by date
   - View analytics
   - Manage managers
   - View detailed daily records
